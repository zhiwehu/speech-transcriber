🎙️ Speech Transcriber

MIT License Python 3.11+

多说话人语音识别系统 - 基于阿里达摩院 FunASR 和 3D-Speaker 技术栈

🚀 快速开始

pip install -r requirements.txt
python3 run.py -i audio.wav -o result.txt

📦 功能特性

  • ✅ 语音识别（ASR）- 阿里云 FunASR
  • ✅ 说话人分离 - 3D-Speaker
  • ✅ 中英文混合识别
  • ✅ 在线/本地两种模式
